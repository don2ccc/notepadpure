# NotepadPure

> "我们有时候真的只是需要一个纯粹的 Notepad，仅此而已。"

**NotepadPure** —— 为 Windows 开发者打造的极简文本编辑器。

---

## 为什么创造 NotepadPure？

在这个软件越来越臃肿的时代，我们被各种功能堆砌的编辑器包围。它们强大，却沉重；它们全能，却复杂。

有时候，我们只是想：
- 快速打开一个配置文件
- 随手记下一段代码片段
- 纯粹地编辑文本，而不被干扰

**NotepadPure 应运而生。**

没有繁复的菜单，没有冗余的插件，没有令人眼花缭乱的配置。只有你和文字，仅此而已。

---

## 核心理念

### 🪶 轻如鸿毛
- 安装包仅数 MB
- 启动瞬间完成
- 内存占用极低

### 🎯 纯粹至简
- 文件夹树状浏览，一目了然
- 多标签页编辑，从容切换
- 零学习成本，开箱即用

### 🛠️ 开发者友好
- 智能编码检测与转换
- 正则表达式查找替换
- Base64 / MD5 / SHA 实用工具
- Markdown 实时预览

---

## 功能一览

| 功能 | 描述 |
|------|------|
| 📁 文件夹浏览 | 左侧树状目录，快速定位文件 |
| 📝 多标签编辑 | 同时打开多个文件，标签式管理 |
| 🔍 查找替换 | 支持正则，支持 `\n` 换行符替换 |
| 🌓 主题切换 | 深色/浅色模式，随心切换 |
| 📊 字符统计 | 实时显示行列与字符数 |
| 🔐 编码工具 | Base64 编解码，MD5/SHA1/SHA256 哈希 |
| 📝 Markdown | 内置预览，所见即所得 |
| 🔄 编码转换 | UTF-8 / GBK / GB2312 自动识别与转换 |

## 下载安装

### 系统要求
- Windows 10/11 64位
- WebView2 Runtime（Windows 11 已内置）

### 安装方式
1. 访问 [Releases](https://github.com/don2ccc/NotepadPure/releases) 页面
2. 下载最新版本的 `NotepadPure_x64.msi`
3. 双击安装，即刻使用

---

## 使用技巧

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + O` | 打开文件夹 |
| `Ctrl + S` | 保存文件 |
| `Ctrl + Shift + S` | 另存为 |
| `Ctrl + F` | 查找（使用编辑器内置功能） |
| `Ctrl + H` | 替换 |
| 双击查找面板标题 | 切换查找/查找替换模式 |
